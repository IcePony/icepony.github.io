    画质类

    OptiFine

    使得原版可调整的地方更多了

    • Optifine是高清补丁，可以使画面更细腻，画面设置分得更细(比如原版中，调节画质只能调流畅或高品质，装了高清补丁后，可以调节单一的树木、水、云等)还具有放大功能，还能使各种贴图完全加载，使纹理衔接在一起(比如无缝玻璃)，总之，装了之后画面会变好看

    LessLag

    禁用纹理动画

    • Basically the mod “disables” a method in base minecraft that “updates” animations of every texture 20 times a second, i obviously just stopped it from updating at all unless it’s one of the blocks included in the config.

      基本上mod会禁用基于Minecraft中的每秒20次更新纹理的动画，我显然只是阻止它进行更新，除非它是配置中包含的块之一。

    优化/修复类

    Betterfps

    通过优化mc的sin/cos函数的算法，来增加帧数

    1. 快速漏斗

    2. 它们在不需要时刻检查物品。只需在其上方放置一个坚固的方块就可以停止寻找要拾取的物品。

    3. 内存预分配

      • 删除10MB的RAM预分配。添加此分配是为了防止玩家以低内存启动游戏，但这种分配一直保持到几乎内存达到极限。这仅适用于使用modpacks的玩家，其中内存可能是个问题。如果你有足够的内存，这对你来说几乎没用。

    4. 快速搜索

      • 创意模式搜索真的没有优化，在大型整合包中，按下文本输入中的任何键都会使游戏卡顿片刻。此举改进实现了智能搜索算法，将旧结果重新用于新搜索。它还可以防止任何非alphadecimal键（例如shift）更新搜索。

        它还支持多线程/异步搜索：它们可以防止搜索时出现滞后峰值

    5. 雾造成了很多卡顿，为什么不添加一个选项来删除它？

    6. 去除光束的透明层。当有很多信标时，它可能会有所帮助。

    Clumps

    • Clumps groups XP orbs together into a single entity to reduce lag when there are many in a small area. Ontop of this, it also makes the player immediately collect the orbs once they touch the player, so you are not stuck with a bunch of orbs in your face.

      Clumps将XP球体组合成一个单独的实体，以减少在小区域中存在许多延迟时的滞后。

      在此之后，它还会让玩家在接触到玩家后立即收集球体，这样你就不会被一堆球体困在脸上。

    FastWorkbench

    • Specifically, it caches the last recipe used, and checks this recipe first before re-scanning the entire registry. This makes shift click crafting operate much faster than before. It also lowers network traffic by sending a packet only when needed instead of whenever a slot in changed in the crafting grid.

      具体来说，它会缓存最后使用的配方，并在重新扫描整个注册表之前先检查此配方。这使得合成比以前更快。它还通过仅在需要时发送数据包,而不是在合成网格中更改物品的位置来降低网络流量。

    FoamFix

    • FoamFix is a mod designed to optimize post-1.7.10 modded Minecraft using simple, targeted optimizations. Observations show that Java heap usage (measured using VisualVM on the main menu) can drop by as much as 50%!

      FoamFix是一个mod，旨在使用简单的，有针对性的优化来优化1.7.10之后的Minecraft。观察显示Java堆使用率（使用主菜单上的VisualVM测量）可以下降多达50％！

    SmothFont

    • This mod draws all fonts (including resource pack’s fonts) smoothly for better readability on all scales. Furthermore, you can use any font installed on your computer.

      此mod可以平滑地绘制所有字体（包括资源包的字体），以便在所有比例下更好地读取。此外，您可以使用计算机上安装的任何字体。

      将这个模组加入游戏中后，一切都加载完毕后，它会遍历材质管理器并删除所有非动态材质所造成的内存空间碎片，以减少内存占用。

    WHAT - Fluid Fix

    • It’s time for technical explanations again. The forge fluid registry has a fatal flaw in that it can change the default fluid on world load, but only one forge bucket will be baked and only one fluid texture will be stitched into the texture atlas. This results in black and purple quilt textures on fluid tanks if two mods add the same fluid (Steam, Oil, Molten Metals, …). This mod doesn’t solve the discrepancy between the world default and the load default, it simply stitches ALL fluid textures to ensure there will be a texture to render even if the world default doesn’t match the load default.

      是时候再次进行技术解释了。Forge液体注册表有一个致命的缺陷，它可以改变世界加载的默认流体，但只有一个Forge桶将被加载，只有一个流体纹理将被缝合到纹理图集中。如果两个mod添加相同的流体（蒸汽，油，熔融金属……），这会在流体罐上产生黑色和紫色的纹理。此mod不能解决改变世界,它只是缝合所有流体纹理，以确保即使世界默认值与加载默认值不匹配也不会有渲染错误纹理。
